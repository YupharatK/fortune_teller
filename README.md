# 🔮 นำโชค Fortune Teller (Client-Server App)

โปรแกรมดูดวงสไตล์ Pixel Art RPG ที่ทำงานด้วยระบบ Client-Server ผ่าน TCP Socket 
ผู้ใช้สามารถกรอกชื่อ วันเกิด และเลือกหมวดหมู่การดูดวง (ความรัก, การเงิน, การเรียน) เพื่อรับคำทำนายสุดแม่นยำจาก "แม่หมอแมวดำนำโชค" พร้อมระบบสร้างผลดวงเป็นไฟล์ PDF!

---

## 🛠️ สิ่งที่ต้องมีก่อนรันโปรแกรม (Prerequisites)

โปรเจกต์นี้เขียนด้วยภาษา **Python** กรุณาตรวจสอบให้แน่ใจว่าติดตั้ง Python ในเครื่องแล้ว (แนะนำเวอร์ชัน 3.8 ขึ้นไป) และติดตั้ง Library ที่จำเป็นดังนี้:

### 1. ติดตั้ง Library
เปิด Terminal หรือ Command Prompt แล้วรันคำสั่งนี้เพื่อติดตั้งแพ็กเกจที่จำเป็นทั้งหมด:
```bash
pip install customtkinter Pillow reportlab

## 🚀 วิธีการรันโปรแกรม (How to Run)

เนื่องจากโปรแกรมนี้ทำงานแบบ **Client-Server** คุณจะต้องเปิดโปรแกรมฝั่ง Server ก่อน แล้วค่อยเปิดฝั่ง Client ให้เชื่อมต่อเข้ามาครับ (หากรันในเครื่องเดียวกัน ให้เปิด Terminal แยกกัน 2 หน้าต่าง)

### ขั้นตอนที่ 1: เปิด Server (พนักงานต้อนรับ & แม่หมอ)
เปิด Terminal หน้าต่างแรกขึ้นมา แล้วใช้คำสั่ง `cd` (Change Directory) เพื่อเข้าไปยังโฟลเดอร์โปรเจกต์ของคุณก่อน:
```bash
cd path/to/your/fortune_teller

python server.py

python client.py
